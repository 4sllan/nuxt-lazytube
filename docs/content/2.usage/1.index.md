After installing the module, you can start using the `<LazyYoutube>` and `<LazyVimeo>` components in your Nuxt application.

## Basic Usage

### YouTube Videos

::prose-pre
---
code: app.vue
filename: app.vue
---

```vue
<template>
  <LazyYoutube src="https://www.youtube.com/watch?v=TcMBFSGVi1c" />
</template>
```
::

### Vimeo Videos

::prose-pre
---
code: app.vue
filename: app.vue
---

```vue
<template>
  <LazyVimeo src="https://player.vimeo.com/video/64654583" />
</template>
```
::

## Supported URL Formats

Both components accept various URL formats:

- **YouTube**: 
  - `https://www.youtube.com/watch?v=VIDEO_ID`
  - `https://youtu.be/VIDEO_ID`
  - `https://www.youtube.com/embed/VIDEO_ID`

- **Vimeo**:
  - `https://vimeo.com/VIDEO_ID`
  - `https://player.vimeo.com/video/VIDEO_ID`

Any additional query parameters in the URL will be automatically included in the iframe.
