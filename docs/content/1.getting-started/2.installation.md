# Installation

Nuxt Lazytube is designed for Nuxt 3 applications. Follow these steps to install and configure the module.

## Install the Module

Use the Nuxt CLI to add the module to your project:

::prose-pre
---
code: npx nuxi@latest module add nuxt-lazytube
filename: Terminal
---

```bash
npx nuxi@latest module add nuxt-lazytube
```
::

## Configure Nuxt

Add the module to your `nuxt.config.ts` (or `nuxt.config.js`) file:

::prose-pre
---
code: nuxt.config.ts
filename: nuxt.config.ts
---

```ts
export default defineNuxtConfig({
  modules: [
    'nuxt-lazytube',
  ],
})
```
::

## Verify Installation

Once installed, you can start using the components in your application. The module automatically registers two components:

- `<LazyYoutube>` - For YouTube videos
- `<LazyVimeo>` - For Vimeo videos
