# Getting Started

Nuxt Lazytube is a Nuxt module that allows you to embed YouTube and Vimeo players easily with lazy loading to save resources and reduce initial load size.

## Why Use Nuxt Lazytube?

Lazy loading videos is essential when performance and page size are important, especially for sites with many embedded videos. For a simple example page with 10 videos, nuxt-lazytube will reduce load time by 7x and memory usage by 2-3x.

## Key Benefits

- **Performance**: Reduces initial load size by ~1.1 MB per video
- **Responsive**: Fully responsive and customizable through props
- **Control**: Provides methods to control (play, stop, pause and reset) embedded videos
- **Customization**: Options to set up custom title and preview of embedded videos
- **Platform Support**: Full support for YouTube and Vimeo
